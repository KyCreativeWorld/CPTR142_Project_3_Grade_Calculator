# CPTR142_Project_3_Grade_Calculator
Our project 3 for CPTR142 Fundamentals of Programming II is a grade calculator. Made by Ky Kendall, Adin Neufeld, and Spencer Shipowick.

**Description**
*Reads from a gradebook txt file.* Then, it gives you a menu to look at a specific class. For any class, you can see your grades on different assignments and exams. You can also calculate the minimum grade you need to get on the final exam to get a desired grade.

**Assumptions**
*Assumes that the user understands how to use the program*, that the user is a good actor, that no security is required, and that the user is using US. grading system.

**How to use**
1. *Run the program.*
2. *Input the name of a gradebook file.*
3. *Input the name of the class you want to view.*
4. *Input the grade you desire in the class.*
5. *Watch as a calculator outputs your results.*
6. *Input another grade or type -1 to exit to the class menu.*
8. *Input another class name or type 'q' or 'quit' to exit the program.*
